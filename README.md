Sales Data ETL Pipeline

Tech Stack:
- Python (pandas)
- PostgreSQL
- Power BI

Description:
Built an end-to-end ETL pipeline that processes retail sales data from CSV,
cleans and loads it into PostgreSQL, and visualizes insights using Power BI.

Pipeline:
CSV → Python → PostgreSQL → Power BI

Key Dashboards:
- Total Sales by Category
- Sales by Region
- Yearly Sales Trend
- Top Products by Sales

Dataset:
Kaggle Superstore (9.7k+ records)
